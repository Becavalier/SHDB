# SHDB

### Description
A simple and lightweight local nosql db for shell script, mainly for data persistence.


### Usage

#### Install:

>* sudo bash ./shdb.sh install

#### How to use:

>* shdb status
>* shdb set [key] [value]
>* shdb get [key]
>* shdb delete [key]
>* shdb uninstall
